二、当前活跃的 API 列表

4. 自动购买改名饰品 (行 476-555)
路由: /auto_buy_renamed_weapon [POST]
功能: 自动搜索并购买改名饰品
请求参数: steamId, spider_config (包含 weapon_id, 最大差价, 查询间隔等)
对应前端:
页面: data_spider/SearchWeaponRename
核心类: StreamWeaponSearcher
5. 自动购买挂件饰品 (行 558-643)
路由: /auto_buy_pendant_weapon [POST]
功能: 自动搜索并购买带挂件的饰品
请求参数: steamId, spider_config
对应前端:
页面: data_spider/SearchPendant
核心类: AutoBuyWeaponWithPendant
6. 停止挂件搜索 (行 646-695)
路由: /stop_pendant_search [POST]
功能: 停止挂件搜索任务
请求参数: config_id
对应前端: data_spider/SearchPendant (停止按钮)
7. 获取挂件搜索进度 (行 698-786)
路由: /get_pendant_search_progress [POST]
功能: 轮询获取挂件搜索任务进度
请求参数: config_id
对应前端: data_spider/SearchPendant (进度显示)
数据来源: ConfigAPI (search_pendant_progress)
8. 实时查询悠悠在售底价 (行 789-932)
路由: /getRealTimeLowestPrice [POST]
功能: 实时查询悠悠有品在售底价
请求参数: steamId, yyypId
返回数据: lowest_price, total_count, commodity_list
对应前端: ItemSearch 页面(价格查询功能)
核心类: selectWeapon.fetch_weapon_list()
9. 获取库存扩展信息 (行 965-1086)
路由: /getInventoryExtendInfo [POST]
功能: 批量获取库存饰品的扩展信息(赔付方式列表)
请求参数: steamId, itemsList
返回数据: normalLeaseCompensationMap, longLeasingModeMap
对应前端:
页面: Inventory/RentFormYYYP (出租表单)
核心类: LentWeapon.get_inventory_extend_info()
10. 上架出租饰品 (行 1092-1206)
路由: /uploadRent [POST]
功能: 上架出租饰品接口
请求参数: steamId, items (包含 assetid, shortRentPrice, longRentPrice, depositPrice等)
返回数据: 上架结果列表 + 成功/失败统计
对应前端:
页面: Inventory/RentFormYYYP (确认上架)
核心类: LentWeapon.upload_rent_items()

五、核心业务流程
1. 出租流程 (Inventory → RentFormYYYP)
1. 选择饰品 → 打开出租表单
2. 调用 /getInventoryExtendInfo → 获取赔付配置
3. 用户填写租金、押金、天数等
4. 调用 /uploadRent → 上架出租
2. 自动爬虫流程 (SearchPendant/SearchWeaponRename)
1. 配置爬虫参数(饰品ID、价格区间等)
2. 调用 /auto_buy_pendant_weapon 或 /auto_buy_renamed_weapon
3. 轮询 /get_pendant_search_progress 获取进度
4. 调用 /stop_pendant_search 停止任务
3. 商品购买流程 (ItemSearch)
1. 搜索商品 → 显示列表
2. 调用 /getRealTimeLowestPrice → 查询实时底价
3. 调用 /quickBuyCommodity → 创建订单
4. 调用 /queryPaymentStatus → 确认支付状态
