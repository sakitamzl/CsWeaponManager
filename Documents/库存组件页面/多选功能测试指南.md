# 多选功能测试指南

## 测试环境准备

### 前置条件
1. 确保已启动后端服务
2. 确保已启动前端服务
3. 浏览器打开 `http://localhost:端口号`
4. 登录系统并进入库存页面

## 功能测试清单

### 1. 多选模式切换测试

#### 测试步骤
1. 进入 `/inventory` 页面
2. 切换到"卡片"显示模式
3. 点击右上角"多选"按钮

#### 预期结果
- ✓ 按钮文字变为"取消多选"
- ✓ 按钮颜色变为橙色
- ✓ 所有卡片左上角出现复选框
- ✓ 卡片可以点击选择

#### 测试步骤（退出）
1. 点击"取消多选"按钮

#### 预期结果
- ✓ 按钮文字变为"多选"
- ✓ 按钮颜色变为灰色
- ✓ 复选框消失
- ✓ 所有选择被清空

---

### 2. 物品选择测试

#### 测试步骤
1. 进入多选模式
2. 点击第一个卡片
3. 点击第二个卡片
4. 再次点击第一个卡片

#### 预期结果
- ✓ 第一次点击：卡片被选中，显示蓝色边框
- ✓ 第二次点击：第二个卡片也被选中
- ✓ 第三次点击：第一个卡片取消选中

#### 边界测试
- 选择0件物品：操作栏不显示
- 选择1件物品：操作栏显示"已选择 1 件物品"
- 选择多件物品：操作栏显示正确数量

---

### 3. 操作栏显示测试

#### 测试步骤
1. 进入多选模式
2. 不选择任何物品

#### 预期结果
- ✓ 操作栏不显示

#### 测试步骤
1. 选择至少1件物品

#### 预期结果
- ✓ 操作栏从底部滑入
- ✓ 显示"已选择 X 件物品"
- ✓ 显示"出售"按钮
- ✓ 显示"出租"按钮
- ✓ 显示"清空选择"按钮

---

### 4. 清空选择测试

#### 测试步骤
1. 选择多件物品
2. 点击"清空选择"按钮

#### 预期结果
- ✓ 所有物品取消选中
- ✓ 操作栏消失
- ✓ 仍然保持在多选模式

---

### 5. 出售弹窗测试

#### 测试步骤（无选择）
1. 进入多选模式但不选择物品
2. 尝试点击"出售"按钮（需要先选择物品）

#### 预期结果
- ✓ 显示警告提示："请先选择要出售的物品"

#### 测试步骤（有选择）
1. 选择1-3件物品
2. 点击"出售"按钮

#### 预期结果
- ✓ 弹出"出售物品"弹窗
- ✓ 显示选中物品列表
- ✓ 每个物品显示图片、名称、购入价
- ✓ 显示价格输入框
- ✓ 显示备注输入框
- ✓ 显示"取消"和"确认"按钮

---

### 6. 出租弹窗测试

#### 测试步骤
1. 选择物品
2. 点击"出租"按钮

#### 预期结果
- ✓ 弹出"出租物品"弹窗
- ✓ 其他功能与出售弹窗相同

1. 不输入价格
2. 点击"确认"按钮

1. 输入 "abc"
2. 点击"确认"按钮

- ✓ 不提交表单

1. 输入价格 "100.50"
2. 输入备注 "测试备注"
3. 点击"确认"按钮

#### 预期结果
- ✓ 显示loading状态
- ✓ 提交成功后显示成功提示
- ✓ 弹窗关闭
- ✓ 选择被清空
- ✓ 退出多选模式

## 自动化测试脚本（未来）

```javascript
// 示例：Cypress 测试脚本
describe('库存多选功能测试', () => {
  beforeEach(() => {
    cy.visit('/inventory')
    cy.get('[data-display-mode="card"]').click()
  })

  it('应该能够进入多选模式', () => {
    cy.contains('多选').click()
    cy.contains('取消多选').should('be.visible')
    cy.get('.select-checkbox').should('exist')
  })

  it('应该能够选择物品', () => {
    cy.contains('多选').click()
    cy.get('.inventory-card').first().click()
    cy.get('.inventory-card.selected').should('have.length', 1)
  })

  it('应该显示操作栏', () => {
    cy.contains('多选').click()
    cy.get('.inventory-card').first().click()
    cy.get('.multi-select-actions').should('be.visible')
    cy.contains('已选择 1 件物品').should('be.visible')
  })

  // 更多测试用例...
})
```
