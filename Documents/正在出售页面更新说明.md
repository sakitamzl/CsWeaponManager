## API 接口需求

### 新增接口

1. **获取悠悠有品账号列表**
   ```
   GET /api/yyyp-accounts
   
   返回格式：
   {
     "success": true,
     "data": [
       {
         "id": 1,
         "name": "账号名称",
         "item_count": 10
       }
     ]
   }
   ```

2. **获取BUFF账号列表**
   ```
   GET /api/buff-accounts
   
   返回格式：
   {
     "success": true,
     "data": [
       {
         "id": 1,
         "name": "账号名称",
         "item_count": 5
       }
     ]
   }
   ```

## 使用说明

### 访问页面
1. 点击左侧导航"正在出售"
2. 默认进入"悠悠有品"子页面
3. 使用二级导航切换到"BUFF"

### 操作流程
1. **选择账号** - 页面加载后自动选择第一个账号
2. **切换账号** - 使用下拉框选择其他账号
3. **查看数据** - 自动加载选中账号的在售商品
4. **筛选商品** - 使用搜索框和筛选器
5. **管理商品** - 改价、下架等操作

### 注意事项
1. 必须先选择账号才能查看数据
2. 切换账号会自动刷新数据
3. 每个平台的账号列表独立管理
4. IGXE平台暂时隐藏，可在代码中恢复
